Editing Basics

Linear vs Non-Linear Editing

Linear Editing: The old way (used with VHS tapes and film). You edit in order from start to finish. If you want to change something in the middle, you have to re-record everything after it. Very limited and time-consuming.

Non-Linear Editing (NLE): The modern way (Premiere Pro, DaVinci Resolve, Final Cut, etc.). Your footage is on a computer, so you can jump to any part, make changes, add effects, and experiment freely without destroying previous work. Unlimited undos and total flexibility.

Today, almost all professional editing is non-linear – it's faster and more creative.

Cutting Techniques

Straight Cut: The simplest – just joining two clips. Used for normal pacing.

Jump Cut: Cutting out part of the same shot (e.g., removing pauses in talking). Creates energy or urgency. Popular in YouTube vlogs.

Match Cut: Cutting from one shot to another where the action or shape matches perfectly (e.g., a door closing cuts to a book closing). Makes transitions feel seamless and artistic.

J-Cut & L-Cut: Audio from the next/previous scene starts before or continues after the video cut. Great for smooth dialogue scenes.

Timeline Management

The timeline is your editing workspace. Import clips, arrange them in order, trim start/end points, and layer effects/audio.

Key tips: Use markers for key moments, group clips for easy moving, and zoom in/out for precision.

Organize your project with bins/folders for footage, audio, and graphics to avoid chaos.

Audio Editing Basics

Sync audio with video, adjust volume levels, add fades to avoid abrupt changes.

Use EQ to balance frequencies (boost bass for depth, cut highs for clarity), and compression to even out loud/soft parts.

Add sound effects or music tracks on separate layers.

Advanced Effects Explained

Blur & Sharpen

Gaussian Blur: Softens the image evenly. Use to create depth of field (background blur), dream sequences, or reduce noise.

Motion Blur: Simulates blur from fast movement. Makes action look more realistic.

Sharpen: Increases edge contrast to make details pop.

Unsharp Mask: Advanced sharpening – creates a blurred copy and subtracts it from the original for cleaner, more controlled results.

Use blur on backgrounds to guide viewer attention to the subject.

Color Effects

Grayscale: Removes all color for dramatic black-and-white look.

Sepia: Adds warm brown tones for vintage/old-film feel.

Colorize: Applies a single color tint to monochrome footage.

Temperature/Tint: Adjusts warm (orange) vs cool (blue) balance to fix lighting or set mood.

Stylize Effects

Glow: Adds soft light around bright areas – great for magical or dreamy scenes.

Emboss: Creates a raised 3D relief effect, often used on titles.

Mosaic: Breaks image into large pixels – good for censorship or abstract art.

Posterize: Reduces number of colors for bold, graphic novel look.

Distort Effects

Warp: Deforms image using a grid – perfect for face morphing or liquid effects.

Twirl: Spirals the image from the center.

Ripple: Creates water-like waves.

Lens Distortion: Simulates real camera lens flaws (barrel or pincushion).

Generate Effects

Gradient: Creates smooth color transitions for backgrounds or overlays.

Lens Flare: Adds realistic light streaks and bursts.

Grid: Overlays lines for alignment or sci-fi HUDs.

Audio Spectrum: Visualizes music as reacting bars/waves – essential for music videos.

Speed Ramping

Change video speed mid-clip: slow-mo for drama, fast-forward for comedy or time-lapse.

Key steps: Set keyframes for speed changes, use easing for smooth ramps, retime audio to match if needed.

Green Screen/Keying

Remove a solid color background (usually green) to composite over new footage.

Use chroma key tools: Sample the color, adjust tolerance, clean edges with matte chokers. Light evenly to avoid spills.

Green for people (less skin tone overlap), blue for objects.

Transitions

Common Transitions

Radial Wipe: Reveals the next scene like a clock hand sweeping.

Linear Wipe: Slides in from one direction (left, right, top, bottom).

Card Wipe: Flips the scene like turning a page or card.

Use simple cuts most of the time – fancy transitions can distract if overused.

Custom Transitions

Create your own using masks, adjustment layers, or effects like displacement maps.

Example: Use a gradient wipe for smooth blends, or animate shapes for unique reveals.

Dissolve & Fade Transitions

Cross Dissolve: Most common – one clip fades out while next fades in. Great for time passage or soft changes.

Fade to Black: Fade to solid black – classic scene end.

Fade to White: Used for flashbacks or dream sequences.

Zoom & Spin Transitions

Scale and rotate one clip out while scaling next in – popular in music videos and trailers.

Use keyframing or built-in effects.

Motion Graphics Basics

Keyframing

The foundation of all animation. You set "keyframes" where properties change (position, scale, rotation, opacity). The software automatically fills in the movement between them.

Example: Keyframe 1 – text off-screen. Keyframe 2 (2 seconds later) – text centered. Result: smooth slide-in animation.

Text Animation

Start simple: fade in, slide from side, scale up.

Advanced: Animate individual letters, use tracking/position per character, add glitch or type-on effects.

Shape Layers

Vector shapes that scale perfectly. Use "Trim Paths" to draw lines on screen, "Repeater" to create patterns.

Great for logos, icons, and abstract backgrounds.

Easing & Timing

Easing makes movement natural: "Ease Out" (starts fast, slows down), "Ease In" (slow start, speeds up), "Bounce" for playful feel.

Good timing = realistic and engaging animation.

Lower Thirds

On-screen text showing names/titles (common in interviews). Combine text + shape background, animate smooth entry/exit.

Keep clean, readable, and consistent with your brand.

Expressions

Code snippets to automate animation (e.g., wiggle for random shake).

Example: wiggle(5,20) – vibrates 5 times per second by 20 units.

3D Motion Graphics

Add depth with 3D layers, cameras, and lights. Animate position in X/Y/Z space.

Use for parallax effects or rotating elements.

Python Project: Simple To-Do List App

Build a console-based to-do list.

tasks = []
def add_task(task): tasks.append(task)
def remove_task(index): del tasks[index]
def view_tasks(): for i, t in enumerate(tasks): print(f"{i+1}. {t}")

while True:
    print("1. Add 2. Remove 3. View 4. Exit")
    choice = input("Choice: ")
    if choice == "1": add_task(input("Task: "))
    elif choice == "2": remove_task(int(input("Index: ")) - 1)
    elif choice == "3": view_tasks()
    elif choice == "4": break

JavaScript Project: Simple Web To-Do List

HTML + JS for a browser to-do app.

<input id="task" type="text"> <button onclick="addTask()">Add</button>
<ul id="list"></ul>
<script>
let tasks = [];
function addTask() {
  let task = document.getElementById("task").value;
  tasks.push(task);
  updateList();
}
function updateList() {
  let list = document.getElementById("list");
  list.innerHTML = "";
  tasks.forEach((t, i) => {
    let li = document.createElement("li");
    li.textContent = t;
    let btn = document.createElement("button");
    btn.textContent = "Remove";
    btn.onclick = () => { tasks.splice(i, 1); updateList(); };
    li.appendChild(btn);
    list.appendChild(li);
  });
}
</script>

C++ Project: Simple Command-Line Game (Guess the Number)

A number guessing game.

#include <iostream>
#include <cstdlib>
#include <ctime>
int main() {
  srand(time(NULL));
  int secret = rand() % 100 + 1;
  int guess;
  while (true) {
    std::cout << "Guess: ";
    std::cin >> guess;
    if (guess == secret) { std::cout << "Correct!" << std::endl; break; }
    else if (guess < secret) { std::cout << "Too low" << std::endl; }
    else { std::cout << "Too high" << std::endl; }
  }
  return 0;
}
